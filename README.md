# BookReader - Тестовое задание

Мобильное приложение для скачивания и чтения книг, реализованное в рамках тестового задания.

## Технологический стек

- **Язык**: Kotlin
- **UI**: Jetpack Compose, Material3
- **Архитектура**: MVI + Clean Architecture
- **DI**: Dagger Hilt
- **Многомодульность**: Feature-based структура
- **Асинхронность**: Coroutines, Flow, StateFlow

## Бэкенд и хранение данных

- **Аутентификация**: Firebase Authentication
- **Метаданные книг**: Firestore
- **Хранение файлов**: Yandex Cloud Storage (S3-совместимое API)
- **Локальное кэширование**: SharedPreferences, File System
- **Навигация**: Navigation Component


## Ключевые особенности реализации

### MVI Архитектура
- Единообразная структура State/Event для всех экранов
- Предсказуемое управление состоянием UI
- Легкая тестируемость и отладка

### Многомодульность
- Изолированные feature-модули
- Четкое разделение ответственности
- Ускорение сборки за счет параллелизации

### Работа с файлами
- Загрузка в Yandex Cloud Storage через AWS S3 SDK
- Локальное кэширование скачанных книг
- Поддержка форматов: TXT, EPUB, PDF

### Навигация
- BottomNavigation с 3 основными разделами
- Графы навигации для feature-модулей
- Глубокая линковка и восстановление состояния

## Функциональность

### Экран авторизации
- Вход по email/password через Firebase Auth
- Валидация полей ввода
- Обработка состояний загрузки и ошибок
- Автологин при повторном запуске

### Библиотека книг
- LazyColumn со списком скачанных книг
- Локальный поиск по названию и автору
- Управление локальными файлами (удаление/скачивание)
- Синхронизация с Firestore

### Загрузка книг
- Выбор файлов через системный файловый менеджер
- Валидация форматов (TXT, EPUB, PDF)
- Загрузка в Yandex Cloud Storage
- Сохранение метаданных в Firestore

### Читатель книг
- Адаптивные UI для разных форматов
- Настройки отображения (размер шрифта, тема, интервалы)
- Сохранение прогресса чтения
- Поддержка светлой/темной темы

### Профиль пользователя
- Просмотр информации аккаунта
- Добавление аватарки
- Выход из системы
